Local-first · agentic · payment-native

The agent
with a wallet.

Bolt is a personal AI agent that holds its own USDC, acts within rules you set, and runs entirely on your machine. One command to a running agent. Nothing hosted but the model.

$ bun run setup

requires bun · macOS / Linux · ~30s to first chat

◈

Its own wallet

Per-persona USDC vaults with on-chain rules — spend caps, time locks, multisig sign-off. The agent moves money only within them.

⚙

Many tools

Connect any MCP server. Each tool is capability-gated and lands on a proof-of-action ledger — you see exactly what it did.

⌂

Local-first

Personas, memory, keys, and ledger live in ~/.vellum on your machine. The only outbound call is to your LLM.

⚡

Quiet by default

Acts autonomously within its limits, surfaces a plain receipt for anything that moves money, and interrupts you only when it matters.

Up and running in one command

The wizard sets up your key, wallet, and first persona — then launches Bolt.

1Install bun if you don't have it — curl -fsSL https://bun.sh/install | bash
2Clone & enter the repo, then run bun run setup
3Answer the wizard — OpenRouter key, wallet, first persona — and it starts the app for you
4Open http://127.0.0.1:8787 and start chatting
